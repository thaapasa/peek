Color and emphasis

Status messages often carry color. A successful build looks different from a failing one, and an informational note sits somewhere quieter. Sometimes a phrase needs a visual highlight so it is impossible to miss when you skim. The aside in the corner says less by saying it softly.

Inline code spans sit in the middle of sentences, and so do Ctrl + K keyboard hints. A gradient headline shows up occasionally when designers feel ambitious.

Typography variants

  • Old text we no longer endorse: use the deprecated API.
  • A correction underlined in red: do not skip the migration.
  • A heading-like phrase in small capitals for emphasis.
  • Subscript and superscript for chemistry and footnotes1.
  • An abbreviation: CSS with a tooltip.
  • A keyboard shortcut combo: Cmd+Shift+P.
  • A sample of program output: Segmentation fault (core dumped).
  • A variable name in math: x < y + 3.
  • Marked text the editor wants to flag: this passage is under review.
  • An inline quotation: computers do exactly what you tell them to.
