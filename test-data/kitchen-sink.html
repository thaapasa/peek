Definition list

Latency
Time between a request being issued and the first byte of a response arriving.
Throughput
Number of requests a system can serve per unit of time, usually under sustained load.
Tail latency
The slow end of the response-time distribution — typically reported at p95, p99, or p999.

Code

An inline call: peek file.json --print. A block of source:

// Compute the nth Fibonacci number iteratively.
fn fib(n: u64) -> u64 {
    let (mut a, mut b) = (0u64, 1u64);
    for _ in 0..n {
        let next = a + b;
        a = b;
        b = next;
    }
    a
}
